## 🩺 Overview

**PetVetPals** is a full-featured **pet health management web application** designed to modernize how pet owners care for their companions.  
Built with **Next.js**, **Tailwind CSS**, and **Redux Toolkit**, it provides an intuitive, mobile-first interface where users can:

- Connect with veterinarians through **live video consultations**
- Track and update **medical, diet, and medication records**
- Receive **real-time reminders** for treatments and wellness activities
- Leverage **AI-driven insights** to make smarter, data-backed pet care decisions

The platform combines a beautiful, responsive frontend with a scalable backend (Express.js + MongoDB) — bringing together health tracking, e-commerce, and intelligent automation in a seamless experience.

## 🧠 Core Features

- 🐶 **Interactive Pet Dashboard**  
  Unified health and wellness hub with tabs for Overview, Health Records, Medications, and Diet & Activity.

- 🤖 **AI Health Assistants**  
  Smart tools that analyze symptoms, suggest possible conditions, generate nutrition plans, and recommend allergy solutions.

- 💊 **Medication Scheduler & Smart Reminders**  
  Multi-time daily reminders with progress tracking, push notifications, and in-app alerts.

- 📹 **Online Vet Consultations**  
  Secure, real-time video consultations between pet owners and veterinarians.

- 🛍️ **Pet Product Marketplace**  
  E-commerce store for pet essentials with future integration for inventory tracking and Stripe payments.

- 🔔 **In-App & Web Push Notifications**  
  Timely reminders for medication, appointments, and daily care routines.

- 📱 **Responsive & Mobile-First UI**  
  Optimized for all screen sizes with smooth animations and accessible design.

- ⚙️ **Scalable Architecture**  
  Modular and production-ready codebase designed for fast iteration and easy feature expansion.

## ⚙️ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/zeroxlab-digital/petvetpals-client.git
cd petvetpals-client
````

### 2. Install Dependencies

```bash
npm install
```

### 3. Set up envvironment variables

```bash
Create a `.env.local` file in the root directory:
NEXT_PUBLIC_API_BASE=what2810ever01284it0123is0101
GOOGLE_CLIENT=whateveritis
# Add other environment variables as needed
```

### 4. Run the Development Server

```bash
npm run dev
```

The app will be live at **[http://localhost:3000](http://localhost:3000)**
